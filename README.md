# hexo-theme-chord

## Installation
Install theme and renderers:

```
npm install hexo-renderer-sass --save
npm install hexo-render-pug --save
npm install hexo-generator-feed --save
```

## Configuration
Default config:

```
logo: ## site logo path
duoshuo: ## duoshuo_shortname
ga: ## Your Google Analytics tracking id
menu:
  - page: home
    directory: /
  - page: archive
    directory: archives/
  - page: about
    directory: about/
  - page: rss
    directory: atom.xml
```
